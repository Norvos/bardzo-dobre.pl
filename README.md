# bardzo-dobre.pl


**bardzo-dobre.pl is a web application written by Express framework and React library. \
This application allows you to order dishes online and provides restaurant management.**

# Technology stack:

**Frontend:**
-	React 16.9
-	Bootstrap 4.0
-	React-router
- Redux

**Backend:**
-	Express 4.17
-	JWT
- Mongo DB database
- Mongoose
